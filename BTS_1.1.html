1. A cosa serve Boostrap?
-Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice.
-Bootstrap include modelli di progettazione basati su HTML e CSS, nonché plug-in JavaScript opzionali.
-Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi.
2. Cosa devi importare per creare una pagina web con bootstrap?
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js">
<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js">
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js">
3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?
Se non vuoi scaricare e hostare Bootstrap 4 da solo, puoi includerlo da un CDN (Content Delivery Network).
MaxCDN fornisce supporto CDN per CSS e JavaScript di Bootstrap. È inoltre necessario includere jQuery.
4. A cosa serve questo attributo del tag meta?
<content="width=device-width, initial-scale=1">
Bootstrap 4 è progettato per rispondere ai dispositivi mobili. Gli stili mobile-first fanno parte del framework principale.
-Il width=device-width imposta la larghezza della pagina per seguire la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo).
-Il initial-scale=1 imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.
5. Che differenza c’è tra Container e Container Fluid?
La classe .container fornisce un contenitore reattivo a larghezza fissa.
La classe .container-fluid fornisce un contenitore reattivo a larghezza fissa, che copre l'intera larghezza della finestra.
